# 金融量化与证券分析常见关键词汇

## 概述

本词汇表涵盖金融量化投资和证券分析领域的核心概念，包括基础指标、技术分析、财务分析、风险指标、量化策略、证券分析、衍生品及组合管理等类别。每个词条包含词汇名称、中英文解释和计算方法。

---

## 一、基础指标类

### 1.1 价格指标

#### 收盘价 (Close Price)
- **解释**: 股票在交易日最后一笔交易的成交价格
- **计算方法**: 取当日最后一笔交易的成交价格，无需计算

#### 开盘价 (Open Price)
- **解释**: 股票在交易日第一笔交易的成交价格
- **计算方法**: 取当日第一笔交易的成交价格，无需计算

#### 最高价 (High Price)
- **解释**: 股票在交易日内的最高成交价格
- **计算方法**: 取当日所有成交价格中的最大值，无需计算

#### 最低价 (Low Price)
- **解释**: 股票在交易日内的最低成交价格
- **计算方法**: 取当日所有成交价格中的最小值，无需计算

#### 成交量 (Volume)
- **解释**: 股票在交易日内的总成交股数
- **计算方法**: 累加当日所有交易的股数，无需计算

#### 成交额 (Turnover/Money)
- **解释**: 股票在交易日内的总成交金额
- **计算方法**: 累加当日所有交易的金额 = 成交价格 × 成交股数

### 1.2 收益率指标

#### 日收益率 (Daily Return)
- **解释**: 股票每日的价格变化百分比
- **计算方法**: `Return = (今日收盘价 - 昨日收盘价) / 昨日收盘价 × 100%`

#### 累计收益率 (Cumulative Return)
- **解释**: 从初始投资到当前的总收益
- **计算方法**: `Cumulative Return = (当前价格 - 初始价格) / 初始价格 × 100%`

#### 年化收益率 (Annualized Return)
- **解释**: 将任意期限的收益率换算为年度收益率
- **计算方法**: `Annualized Return = (1 + Total Return)^(252/N) - 1`，其中 N 为交易天数

#### 对数收益率 (Log Return)
- **解释**: 使用对数计算的收益率，便于数学运算
- **计算方法**: `Log Return = ln(今日收盘价 / 昨日收盘价)`

## 二、技术分析类

### 2.1 移动平均线

#### 简单移动平均线 (SMA - Simple Moving Average)
- **解释**: 过去 N 天收盘价的算术平均值
- **计算方法**: `SMA(N) = (C1 + C2 + ... + CN) / N`，其中 C 为收盘价

#### 指数移动平均线 (EMA - Exponential Moving Average)
- **解释**: 近期价格权重更大的移动平均线
- **计算方法**: `EMA_t = α × Price_t + (1-α) × EMA_{t-1}`，其中 α = 2/(N+1)

#### 加权移动平均线 (WMA - Weighted Moving Average)
- **解释**: 线性加权计算的移动平均线
- **计算方法**: `WMA = (W1×P1 + W2×P2 + ... + WN×PN) / (W1 + W2 + ... + WN)`

### 2.2 趋势指标

#### MACD (Moving Average Convergence Divergence)
- **解释**: 反映价格趋势变化的指标，由快线、慢线和柱状图组成
- **计算方法**:
  - DIF = EMA(12) - EMA(26)
  - DEA = EMA(DIF, 9)
  - MACD柱 = 2 × (DIF - DEA)

#### RSI (Relative Strength Index)
- **解释**: 衡量价格变动速度和幅度的超买超卖指标
- **计算方法**:
  - 上涨平均 = N日内上涨幅度之和 / N
  - 下跌平均 = N日内下跌幅度绝对值之和 / N
  - RS = 上涨平均 / 下跌平均
  - RSI = 100 - 100/(1+RS)
  - 常用参数 N=14

#### 布林带 (Bollinger Bands)
- **解释**: 由中轨和上下轨组成的趋势通道指标
- **计算方法**:
  - 中轨 = MA(20)
  - 上轨 = 中轨 + 2 × 标准差(20)
  - 下轨 = 中轨 - 2 × 标准差(20)

### 2.3 动量指标

#### 动量指标 (Momentum)
- **解释**: 反映价格变化动能的指标
- **计算方法**: `Momentum = 今日收盘价 - N日前收盘价`

#### 变动率指标 (ROC - Rate of Change)
- **解释**: 价格变化的百分比速率
- **计算方法**: `ROC = (今日收盘价 - N日前收盘价) / N日前收盘价 × 100%`

## 三、财务分析类

### 3.1 估值指标

#### 市盈率 (PE Ratio - Price to Earnings Ratio)
- **解释**: 股票价格与每股收益的比率，反映投资回本周期
- **计算方法**: `PE = 股票价格 / 每股收益(EPS)` 或 `PE = 总市值 / 净利润`

#### 市净率 (PB Ratio - Price to Book Ratio)
- **解释**: 股票价格与账面价值的比率
- **计算方法**: `PB = 股票价格 / 每股净资产` 或 `PB = 总市值 / 净资产`

#### 市销率 (PS Ratio - Price to Sales Ratio)
- **解释**: 股票价格与营业收入的比率
- **计算方法**: `PS = 股票价格 / 每股营业收入` 或 `PS = 总市值 / 营业收入`

#### 股息率 (Dividend Yield)
- **解释**: 股票每年分红与股价的比率
- **计算方法**: `Dividend Yield = 每股分红 / 股票价格 × 100%`

### 3.2 盈利能力指标

#### 净资产收益率 (ROE - Return on Equity)
- **解释**: 衡量股东权益收益水平的指标
- **计算方法**: `ROE = 净利润 / 股东权益 × 100%`

#### 总资产收益率 (ROA - Return on Assets)
- **解释**: 衡量企业运用资产盈利能力的指标
- **计算方法**: `ROA = 净利润 / 总资产 × 100%`

#### 毛利率 (Gross Margin)
- **解释**: 销售收入减去成本后的利润比率
- **计算方法**: `毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%`

#### 净利率 (Net Margin)
- **解释**: 净利润占营业收入的比例
- **计算方法**: `净利率 = 净利润 / 营业收入 × 100%`

### 3.3 偿债能力指标

#### 资产负债率 (Debt to Asset Ratio)
- **解释**: 负债总额占资产总额的比率
- **计算方法**: `资产负债率 = 负债总额 / 资产总额 × 100%`

#### 流动比率 (Current Ratio)
- **解释**: 衡量企业短期偿债能力的指标
- **计算方法**: `流动比率 = 流动资产 / 流动负债`

#### 速动比率 (Quick Ratio)
- **解释**: 剔除存货后的短期偿债能力指标
- **计算方法**: `速动比率 = (流动资产 - 存货) / 流动负债`

## 四、风险指标类

### 4.1 波动率指标

#### 历史波动率 (Historical Volatility)
- **解释**: 过去收益率的标准差，反映价格波动程度
- **计算方法**:
  - 计算日收益率序列
  - `Volatility = √(Σ(r_i - r̄)² / N) × √252`
  - 其中 r_i 为日收益率，r̄ 为平均收益率，N 为天数

#### 隐含波动率 (IV - Implied Volatility)
- **解释**: 由期权价格反推的波动率，反映市场对未来波动的预期
- **计算方法**: 使用 Black-Scholes 模型反推，需迭代求解

### 4.2 风险调整收益指标

#### 夏普比率 (Sharpe Ratio)
- **解释**: 衡量承担单位风险获得的超额收益
- **计算方法**: `Sharpe = (投资组合收益率 - 无风险收益率) / 投资组合标准差`

#### 索提诺比率 (Sortino Ratio)
- **解释**: 只考虑下行风险的收益风险比
- **计算方法**: `Sortino = (投资组合收益率 - 目标收益率) / 下行标准差`

#### 卡玛比率 (Calmar Ratio)
- **解释**: 收益与最大回撤的比率
- **计算方法**: `Calmar = 年化收益率 / 最大回撤`

#### 特雷诺比率 (Treynor Ratio)
- **解释**: 衡量系统性风险调整后的收益
- **计算方法**: `Treynor = (投资组合收益率 - 无风险收益率) / Beta`

### 4.3 风险度量指标

#### 最大回撤 (Maximum Drawdown)
- **解释**: 从历史最高点到最低点的最大跌幅
- **计算方法**:
  - 记录每个时点的历史高点
  - `Drawdown = (历史高点 - 当前价值) / 历史高点`
  - `Max Drawdown = max(Drawdown)`

#### VaR (Value at Risk)
- **解释**: 在给定置信水平下的最大可能损失
- **计算方法**: `VaR = 期望收益 - 分位数 × 标准差`
  - 95%置信度下使用1.65倍标准差
  - 99%置信度下使用2.33倍标准差

#### Beta
- **解释**: 衡量股票相对于市场系统性风险的指标
- **计算方法**: `Beta = Cov(股票收益, 市场收益) / Var(市场收益)`

## 五、量化策略类

### 5.1 阿尔法与贝塔

#### Alpha (阿尔法)
- **解释**: 投资组合超越基准的超额收益
- **计算方法**: `Alpha = 实际收益 - Beta × 市场收益`

#### Beta (贝塔)
- **解释**: 系统性风险敞口，衡量相对于市场的波动
- **计算方法**: `Beta = Cov(股票收益, 市场收益) / Var(市场收益)`

### 5.2 因子模型

#### Fama-French 三因子
- **解释**: 由市场因子、规模因子、价值因子解释收益的模型
- **计算方法**:
  - R_i - R_f = α + β₁(R_m - R_f) + β₂SMB + β₃HML + ε
  - SMB = 小市值减大市值收益
  - HMB = 高账面市值比减低账面市值比收益

#### CAPM 模型
- **解释**: 资本资产定价模型
- **计算方法**: `E(R_i) = R_f + β_i(E(R_m) - R_f)`

### 5.3 策略评估指标

#### 信息比率 (Information Ratio)
- **解释**: 主动收益与跟踪误差的比率
- **计算方法**: `IR = 超额收益 / 跟踪误差`

#### 胜率 (Win Rate)
- **解释**: 盈利交易占总交易的比例
- **计算方法**: `Win Rate = 盈利交易数 / 总交易数 × 100%`

#### 盈亏比 (Profit Loss Ratio)
- **解释**: 平均盈利与平均亏损的比值
- **计算方法**: `盈亏比 = 平均盈利金额 / 平均亏损金额`

#### 复利年化收益率 (CAGR - Compound Annual Growth Rate)
- **解释**: 复合年均增长率
- **计算方法**: `CAGR = (期末价值 / 期初价值)^(1/年数) - 1`

## 六、证券分析类

### 6.1 股票筛选指标

#### 每股收益 (EPS - Earnings Per Share)
- **解释**: 每股创造的净利润
- **计算方法**: `EPS = 净利润 / 流通股数`

#### 每股净资产 (BPS - Book Value Per Share)
- **解释**: 每股对应的净资产
- **计算方法**: `BPS = 净资产 / 流通股数`

#### 每股营业收入 (SPS - Sales Per Share)
- **解释**: 每股创造的营业收入
- **计算方法**: `SPS = 营业收入 / 流通股数`

### 6.2 增长指标

#### 营业收入增长率
- **解释**: 营业收入同比或环比增长的速度
- **计算方法**: `增长率 = (本期营业收入 - 上期营业收入) / 上期营业收入 × 100%`

#### 净利润增长率
- **解释**: 净利润同比或环比增长的速度
- **计算方法**: `增长率 = (本期净利润 - 上期净利润) / 上期净利润 × 100%`

#### 每股收益增长率
- **解释**: 每股收益的增长速度
- **计算方法**: `增长率 = (本期EPS - 上期EPS) / 上期EPS × 100%`

### 6.3 营运能力指标

#### 存货周转率
- **解释**: 存货周转的速度
- **计算方法**: `存货周转率 = 营业成本 / 平均存货`

#### 应收账款周转率
- **解释**: 应收账款回收的速度
- **计算方法**: `应收账款周转率 = 营业收入 / 平均应收账款`

#### 总资产周转率
- **解释**: 资产周转的效率
- **计算方法**: `总资产周转率 = 营业收入 / 平均总资产`

## 七、衍生品分析类

### 7.1 期权指标

#### 期权 Delta
- **解释**: 期权价格对标的价格变化的敏感度
- **计算方法**: `Delta = ∂期权价格 / ∂标的价格`

#### 期权 Gamma
- **解释**: Delta 对标的价格变化的敏感度
- **计算方法**: `Gamma = ∂²期权价格 / ∂标的价格²`

#### 期权 Vega
- **解释**: 期权价格对波动率的敏感度
- **计算方法**: `Vega = ∂期权价格 / ∂波动率`

#### 期权 Theta
- **解释**: 期权价格对时间衰减的敏感度
- **计算方法**: `Theta = ∂期权价格 / ∂时间`

### 7.2 期货指标

#### 基差 (Basis)
- **解释**: 现货价格与期货价格的差
- **计算方法**: `基差 = 现货价格 - 期货价格`

#### 升水/贴水
- **解释**: 期货价格高于(升水)或低于(贴水)现货价格
- **计算方法**: `升贴水 = 期货价格 - 现货价格`

## 八、组合管理类

### 8.1 资产配置

#### 协方差矩阵
- **解释**: 资产收益之间的相关性
- **计算方法**: `Cov(i,j) = E[(R_i - μ_i)(R_j - μ_j)]`

#### 相关系数
- **解释**: 两个资产收益的相关程度
- **计算方法**: `ρ = Cov(i,j) / (σ_i × σ_j)`

#### 最优组合权重
- **解释**: 在给定约束下使组合收益最大或风险最小的权重
- **计算方法**: 使用均值-方差优化求解

### 8.2 再平衡指标

#### 组合周转率
- **解释**: 组合持仓变化的频率
- **计算方法**: `周转率 = 买入卖出交易量 / 组合市值`

#### 交易成本
- **解释**: 交易产生的佣金、印花税、滑点等成本
- **计算方法**: `总成本 = 佣金 + 印花税 + 滑点`
