# 绩效指标计算

## 常用绩效指标

| 指标 | 说明 | 合格标准 | 优秀标准 |
|------|------|----------|----------|
| 年化收益率 | 年度平均收益 | > 10% | > 20% |
| 夏普比率 | 风险调整收益 | > 0.5 | > 1.0 |
| 最大回撤 | 最大亏损幅度 | < 20% | < 10% |
| 胜率 | 盈利交易比例 | > 50% | > 60% |
| 卡玛比率 | 收益/最大回撤 | > 1.0 | > 2.0 |
| 波动率 | 收益波动程度 | 越低越好 | < 15% |
| 索提诺比率 | 下行风险调整收益 | > 0.5 | > 1.0 |

## 回测结果字段

回测成功时返回以下字段：

| 字段 | 说明 |
|------|------|
| total_return | 总收益率 |
| annual_return | 年化收益率 |
| sharpe_ratio | 夏普比率 |
| max_drawdown | 最大回撤 |
| win_rate | 胜率 |
| trading_days | 交易天数 |
| volatility | 波动率 |
| calmar_ratio | 卡玛比率 |
| sortino_ratio | 索提诺比率 |

## 回测失败时的错误字段

| 字段 | 说明 |
|------|------|
| error | 错误信息 |
| error_category | 错误类别：client_error / code_logic / engine_error |
| error_detail | 错误详情 |

## 策略评估标准

### 合格策略
- 年化收益率 > 10%
- 夏普比率 > 0.5
- 最大回撤 < 20%

### 优秀策略
- 年化收益率 > 20%
- 夏普比率 > 1.0
- 最大回撤 < 10%
- 卡玛比率 > 2.0
